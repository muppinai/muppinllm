# Changelog

All notable changes to MuppinLLM will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-15

### Added
- Initial release of MuppinLLM
- Technical analysis module with RSI, MACD, Bollinger Bands, Moving Averages
- Fundamental analysis module with liquidity, volume, and token age metrics
- Sentiment analysis module with price momentum and community activity
- DexScreener API integration for Solana DEX data
- CoinGecko API integration for additional token metadata
- AI-powered analysis using GPT-5.2 via Emergent LLM
- Command-line interface (CLI) for easy token analysis
- Python API for programmatic access
- Support for analyzing multiple tokens
- JSON export functionality
- Comprehensive verdict system (BULLISH to BEARISH scale)

### Features
- Real-time Solana token data from DexScreener
- Multiple technical indicators calculation
- Weighted scoring system for final verdict
- AI-generated summaries and recommendations
- Risk factors and opportunities identification
- Market sentiment overview
